What is Windows Azure?

Windows Azure is an open and flexible cloud platform that enables you to quickly build, deploy and manage applications across a global network of Microsoft-managed datacenters.

What is Sencha Touch?

Sencha Touch is a high-performance, HTML5 mobile app platform with over 50 built-in components, themes for every mobile platform, and a built-in MVC system.

What is Sencha Touch Extensions for Windows Azure?

Sencha Touch Extensions for Windows Azure is an API designed to provide an easy way to develop Sencha Touch applications that can access the Windows Azure APIs using JavaScript.
